## How to use

1. Clone this repository:
   ```bash
   git clone https://github.com/zolyniakmilosz222/first-steps.git
   ```
2. Explore the code and folders.
3. Run Python scripts, e.g.:
   ```bash
   python my-python-tools/tamagotchi.py
   ```
